# 梅菲斯特（Mephisto）文档

欢迎来到梅菲斯特的文档中心。

梅菲斯特是一个 **长线叙事引擎**。它通过纯文本的 `.meph` 契约文件，将规则引擎与大模型（LLM）结合，驱动角色在漫长的叙事中保持连贯性。

## 快速导航

- **新手入门**：阅读 [快速开始](../README.md) 运行你的第一个故事。
- **语法手册**：查看 [SYNTAX.md](./SYNTAX.md) 了解 `.meph` 文件的所有写法。
- **规则深度解析**：查看 [RULES.md](./RULES.md) 掌握条件逻辑、骰子和动作。
- **实战示例**：查看 [契约文件模板](../data/sample.meph) 了解如何编写一个完整的契约文件。

## 核心理念

> **你写下角色的灵魂，梅菲斯特让它活过来，然后看它会走向何方。**

- **契约先行**：用 `【】` 定义区块，用 `-` 定义状态，用 `[规则]` 定义行为。
- **命运视角**：你是“命运”（叙事的推动者），输入指令，角色在规则与 LLM 的共同驱动下行动。
- **透明反馈**：规则触发时提供清晰的注入信息（支持 `-quiet` 模式隐藏）。
